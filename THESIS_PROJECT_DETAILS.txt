PROJECT DESCRIPTION
================================================================================
Attendify is a comprehensive, role-based attendance management system designed 
to streamline attendance tracking in educational institutions. The system 
provides real-time analytics, automated notifications, and a modern user 
interface with dark mode support.

KEY FEATURES
================================================================================
1. Role-Based Access Control (RBAC)
   - Admin: Complete system management
   - Teacher: Class and attendance management
   - Student: View attendance records and analytics

2. Real-Time Dashboard
   - Interactive charts using Chart.js
   - Live attendance statistics
   - Performance analytics

3. Modern User Interface
   - Tailwind CSS 4 for styling
   - Dark mode support
   - Responsive design for all devices
   - Alpine.js for interactive components

4. Attendance Management
   - Multiple marking options (Present, Absent, Late)
   - Bulk attendance marking
   - Session-based tracking
   - Historical records

5. Notification System
   - Real-time notifications
   - Email notifications via Gmail SMTP
   - In-app notification panel
   - Attendance alerts

6. Reports & Analytics
   - Attendance percentage calculations
   - Export functionality
   - Visual charts and graphs
   - Performance tracking

7. Academic Structure
   - Subject management
   - Class session tracking
   - Academic term support
   - Student enrollment system

DATABASE ARCHITECTURE
================================================================================

Core Tables:
1. users - User authentication and role management
2. students - Student profile information
3. teachers - Teacher profile information
4. subjects - Course/subject management
5. academic_terms - Academic year/semester management
6. class_sessions - Individual class sessions
7. attendances - Attendance records
8. notifications - System notifications

Key Relationships:
- User -> Student/Teacher (One-to-One)
- Teacher -> Subjects (One-to-Many)
- Student -> Subjects (Many-to-Many)
- ClassSession -> Attendances (One-to-Many)
- Student -> Attendances (One-to-Many)

INSTALLATION & SETUP
================================================================================
Prerequisites:
- PHP 8.2+
- Composer
- Node.js & npm
- PostgreSQL 18

Local Development:
1. Clone repository
2. Run: composer install
3. Run: npm install
4. Configure .env file
5. Run: php artisan migrate
6. Run: php artisan db:seed
7. Run: npm run build
8. Run: php artisan serve

Production Deployment:
1. Push to GitHub repository
2. Connect to Render.com
3. Configure environment variables
4. Deploy automatically via render.yaml
